# 🌡️ Temperature Converter & 🎲 Number Guessing Game

This repository contains two beginner-friendly **Java console projects** that demonstrate problem-solving and core programming concepts.

---

## 📌 Projects

### 1️⃣ Temperature Converter
- Converts between **Celsius**, **Fahrenheit**, and **Kelvin**.  
- Demonstrates:
  - Methods
  - Scanner for input
  - Conditional statements  

✅ Example:  
25.00 °C = 77.00 °F
25.00 °C = 298.15 K




---

### 2️⃣ Number Guessing Game
- Random number generated between **1–100**  
- User has **10 attempts** to guess  
- Provides hints: `"Too High"`, `"Too Low"`  
- Ends with **Congratulations 🎉** or shows correct number  

✅ Example:  
=== Number Guessing Game ===
Guess the number between 1 and 100!
You have 10 attempts.
Enter your guess: 50
Too low! Attempts left: 9
